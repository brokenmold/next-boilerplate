Next Boilerplate
=====================
A barebones NextJS boilerplate


## Installation
```
git clone https://github.com/brokenmold/next-boilerplate.git
cd next-boilerplate
npm install
```

## Development
`npm run dev`

Development port set to `4180` by default.
Change this in `package.json` lock file.

## Build
`npm build`

## Production
`npm start`
